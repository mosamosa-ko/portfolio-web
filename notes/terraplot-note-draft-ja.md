# 歩くことを地図の遊びに変える。GPS 陣取りゲーム「Terraplot」を作っています

こんにちは。山崎 康（Ko Yamasaki）です。  
コンピュータサイエンス、位置情報、グラフデータ、インタラクティブなプロダクト設計に関心があります。

今取り組んでいるプロジェクトのひとつが、GPS 陣取りゲームの **Terraplot** です。  
Terraplot は、現実の場所を歩くことで地図上のセルを獲得し、移動そのものを遊びに変えることを目指したプロダクトです。

## なぜ Terraplot を作っているのか

地図アプリは多くありますが、移動を「遊び」として捉え直す体験にはまだ余地があると感じています。  
歩く、寄り道する、知らない道に入る、同じ場所でも時間帯で印象が変わる。そうした身体的な移動の面白さを、ただのログではなく、地図上の変化として見せたいと思いました。

Terraplot では、位置情報を取得して、地図上でセルの状態を変化させます。  
プレイヤーの移動が地図上に痕跡として残り、自分の行動が territory として可視化される設計です。

## どんな技術を使っているか

- SwiftUI
- Mapbox
- Firebase
- GPS / Location Services

技術的には、位置更新、地図描画、セル状態の表現、プロダクトの体験設計をどう一体化するかがポイントでした。  
単に位置情報を表示するだけではなく、歩くことに意味が生まれるようなゲームループを考えています。

## このプロジェクトで考えていること

Terraplot は、ゲームであると同時に、位置情報プロダクトの実験でもあります。

- 位置情報をどう体験価値に変えるか
- 地図 UI の上でルールをどう自然に見せるか
- 「移動したくなる」インセンティブをどう設計するか
- 現実世界とデジタルの境界をどう軽やかにつなぐか

こうした問いを、実際にプロダクトとして形にしながら確かめています。
